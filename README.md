# Install :

```
git clone https://github.com/Toufa7/Minishell_Tester.git
cd Minishell_Tester
chmod +x tester.sh
./tester.sh
```

# Modes :

./tester -ecpuur

-e : Echo

-c : Cd

-p : Pipes

-u : Unset

-r : Errors
