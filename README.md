<img src="Ress/exit.png" width="1000"/>


# Install :

```
git clone https://github.com/Toufa7/Minishell_Tester.git
cd Minishell_Tester
chmod +x tester.sh
./tester.sh -options
```

# Synopsis :

./Tester [-h|E|p|x|e]

 The following options are available:
 
- -h        Help
- -E        Echo
- -p        Pipes
- -x        Exit
- -e        Export
